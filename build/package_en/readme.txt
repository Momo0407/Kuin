1. Introduction
-------------------------------------------------------------------------------

    Welcome to Kuin, a programming language created by Kuina-chan.
    Kuin is an easy and fast practical programming language.

    If you are new to Kuin, please follow
    the tutorial (https://kuina.ch/en/kuin/tutorial_exe1).

    Also, see the Kuin documentation (https://kuina.ch/en/kuin)
    for more information on Kuin.
    You can also open Kuin documentation by clicking "Help > View Help"
    from the Kuin Editor menu or by pressing the F1 key on your keyboard.

-------------------------------------------------------------------------------
2. Operating Environment
-------------------------------------------------------------------------------

    Kuin and Kuin-produced programs can run on 64-bit Windows, and can also
    generate C++ and JavaScript source code, in which case they will run in
    a variety of environments.
