1. Introduction
-------------------------------------------------------------------------------

    Welcome to Kuin, a programming language created by Kuina-chan.
    Kuin is an easy and fast practical programming language.

    This zip contains the source code only.
    The compiler and editor for Windows and a set of examples can be downloaded
    from the download page (https://kuina.ch/en/kuin/download).

    Also, see the Kuin documentation (https://kuina.ch/en/kuin)
    for more information on Kuin.

-------------------------------------------------------------------------------
2. Operating Environment
-------------------------------------------------------------------------------

    Kuin can generate C++ and JavaScript source code as well as executable
    files that run on 64-bit Windows, and these can be run in a variety of
    environments.
